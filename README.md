# EF Code First Demo

Первое приложение с Entity Framework 6 используя подход Code First.

## Описание
Проект демонстрирует базовую работу с Entity Framework Code First:
- Создание моделей данных
- Работа с DbContext  
- Автоматическое создание БД

## Технологии
- .NET Framework 4.7.2
- Entity Framework 6
- SQL Server

## Заметка
Connection string содержит тестовые данные для локальной БД.
